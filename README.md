# Mood2Movie
A content-based movie recommendation system built using Machine Learning that suggests movies based on user preferences such as genre, cast, language, and storyline similarity.

This project uses natural language processing and vectorization techniques to compute similarity between movies and deliver personalized recommendations.

 _FEATURES_
  -Content-based filtering using similarity scores
  -Movie filtering by genre, language, and cast
  -Clean and interactive Streamlit web interface
  -Fast and efficient recommendation engine
  -Preprocessed dataset stored using pickle for quick loading

_TECH STACK_
  -Python
  -Pandas & NumPy
  -Scikit-learn
  -Streamlit
  -NLP (CountVectorizer / TF-IDF)

_HOW IT WORKS_
  -Movie data is preprocessed and cleaned
  -Features like genres, cast, and overview are combined
  -Text is vectorized to create feature vectors
  -Cosine similarity is used to find similar movies
  -Top recommendations are displayed in the UI

_GOAL_
To provide users with INTELLIGENT and PERSONALISED movie recommendations using machine learning techniques.
